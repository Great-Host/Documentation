## Overview
SFTP (Secure File Transfer Protocol) provides a secure way to access your server's files using a desktop client. This is recommended for uploading large files, managing many folders, or when you need more robust file management than the web interface provides.

<img src="/img/GameServer/11.png" alt="SFTP Connection Details" width="100%">

## Connection Details
On the SFTP tab, you will find the necessary information to connect:
- **Protocol**: `sftp://`
- **Host**: The server address (e.g., `play-es-2333-sftp.greathost.es`).
- **Port**: The connection port (e.g., `11111`).
- **Username**: Your specific SFTP username.
- **Password**: The same password you use to log in to the panel.

## Recommended SFTP Clients
We recommend using one of the following trusted clients to connect to your server.

### FileZilla
<img src="/img/sftp-clients/FileZilla.png" alt="FileZilla Logo" width="50" style="vertical-align: middle; margin-right: 10px;"> **FileZilla**
- **Description**: Free, cross-platform FTP client with SFTP support.
- **Best for**: Users who want a reliable, standard FTP experience.
- [Download FileZilla](https://filezilla-project.org/)

### WinSCP
<img src="/img/sftp-clients/WinSCP.png" alt="WinSCP Logo" width="50" style="vertical-align: middle; margin-right: 10px;"> **WinSCP**
- **Description**: Popular Windows SFTP client with a graphical interface.
- **Best for**: Windows users who prefer a file explorer-like interface.
- [Download WinSCP](https://winscp.net/eng/download.php)

### Cyberduck
<img src="/img/sftp-clients/Cyberduck.png" alt="Cyberduck Logo" width="50" style="vertical-align: middle; margin-right: 10px;"> **Cyberduck**
- **Description**: Free client for Mac and Windows with cloud storage support.
- **Best for**: Mac users or those connecting to various cloud services.
- [Download Cyberduck](https://cyberduck.io/)

### Command Line
For advanced users, you can use the built-in SFTP command in your terminal or command prompt:

```bash
sftp -P [PORT] [USERNAME]@[HOST]
```
*Example:* `sftp -P 2022 user.12345678@sftp.greathost.es`
