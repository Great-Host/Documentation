# ⚙️ Server Settings Guide

## Overview
The **Settings** tab allows you to configure essential aspects of your server, including its name, startup command, version, and more.

<img src="/img/GameServer/16.png" alt="Server Settings Overview" width="100%">

## Server Details
Here you can change the display name of your server. This is the name that appears in your server list on the panel.
1.  Enter the new name in the **Server Name** field.
2.  Click **Save Changes**.

## Startup Configuration
This section shows the command that runs when your server starts.
- **Startup Command**: Displays the Java arguments and jar file being executed.
- **Docker Image**: Select the Java version your server needs (e.g., Java 8, 11, 17, 21).

<img src="/img/GameServer/17.png" alt="Startup Configuration" width="100%">

To change the Java version:
1.  Click on the **Docker Image** dropdown.
2.  Select the appropriate version (e.g., `Java 21` for newer Minecraft versions).
3.  The server may need a restart for changes to take effect.

## Environment Variables
This area allows you to update the server version or jar file name.
- **Minecraft Version**: Specify the version you want to download (e.g., `latest` or `1.20.1`).
- **Server Jar File**: The name of the jar file to run (usually `server.jar`).
- **Build Number**: For specific builds of software like Paper.

<img src="/img/GameServer/18.png" alt="Environment Variables" width="100%">

## Danger Zone
This section contains destructive actions. **Proceed with caution.**

<img src="/img/GameServer/19.png" alt="Danger Zone - Reinstall Server" width="100%">

### Reinstall Server
- **Action**: Completely wipes your server and reinstalls the original server software.
- **Warning**: **All files will be permanently deleted.** This cannot be undone.
- **How to use**: Click the red **Reinstall Server** button only if you want to start over from scratch.
