## Overview
Protecting your project data is crucial. The IDE's **Backups** system allows you to create snapshots of your entire workspace, ensuring you never lose code due to accidental deletions or errors.

<img src="/img/IDE/8.png" alt="Backups List Interface" width="100%">

## Creating a Backup
1.  Go to the **Backups** tab.
2.  Click the **New Backup** button.
3.  The system will start compressing your files. You can see the progress status change from "In Progress" to "Completed".

<img src="/img/IDE/9.png" alt="Backup Creation Progress" width="100%">

## Restoring a Backup
If you need to revert your project to a previous state:
1.  Locate the backup you want to use.
2.  Click the **Restore** icon (counter-clockwise arrow).
3.  A warning modal will appear.

<img src="/img/IDE/10.png" alt="Restore Backup Confirmation" width="100%">

### Important Warnings
- **Overwrite**: Restoring a backup will delete all current files and replace them with the backup content.
- **Confirmation**: You must type **RESTORE** to confirm this destructive action.
