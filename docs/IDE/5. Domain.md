## Overview
The **Domain** tab allows you to connect a custom domain (e.g., `web.example.com`) to your project. This is especially useful if you are hosting a web application or an API and want a professional-looking URL.

<img src="/img/IDE/11.png" alt="Custom Domain Configuration" width="100%">

## Setting up a Custom Domain
1.  **Enter Domain**: Type your desired subdomain or domain in the **Domain Name** field (e.g., `prueba5.greathost.es`).
2.  **DNS Configuration**: You will see a "Required DNS Configuration" box.
    *   **Type**: `CNAME`
    *   **Name**: `your-subdomain` (or `@` for root domain)
    *   **Value**: `control.greathost.es` (or the value shown in your panel)
3.  **Check Domain**: Click the **Check Domain** button to verify if the DNS records are propagating correctly.
4.  **Request Verification**: Once the DNS is verified, click **Request Verification** to finalize the setup.

> **Note:** DNS changes can take anywhere from a few minutes to 48 hours to propagate globally.
